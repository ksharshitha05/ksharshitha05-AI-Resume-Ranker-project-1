# AI Resume Screening & Ranking System

An intelligent resume screening system that automatically analyzes and ranks resumes based on a given job description using Natural Language Processing (NLP) and semantic similarity.

This project helps recruiters quickly identify the most relevant candidates by comparing resumes with job requirements.

---

## Project Overview

Recruiters often receive hundreds of resumes for a single job role. Manually reviewing each resume is time-consuming and inefficient.

This project solves the problem by building an **AI-powered Resume Ranker** that:

- Extracts information from resumes
- Identifies candidate skills
- Compares resumes with job descriptions
- Calculates similarity scores
- Ranks candidates automatically

---

## Features

- Resume text extraction from PDF files
- Automatic skill detection using NLP
- Semantic similarity using transformer embeddings
- Skill match percentage calculation
- Candidate email and phone extraction
- Resume ranking based on final score
- Interactive dashboard for analysis
- Graph visualization of candidate scores

## Project Architecture

1. User enters a job description.
2. User uploads multiple resumes in PDF format.
3. System extracts text from resumes.
4. NLP model detects skills in resumes and job description.
5. Sentence transformer generates semantic embeddings.
6. Cosine similarity calculates resume relevance.
7. Skill matching score is calculated.
8. Final ranking score is generated.
9. Results are displayed in an interactive dashboard.
